1. 프로젝트 주제
패션 코디 추천 앱 Fitzy
본 프로젝트는 사용자가 업로드한 옷 이미지를 바탕으로, AI모델을 활용하여 자동으로 옷
패션을 탐지하고, 색상과 스타일을 분석하여 최적의 코디를 추천해주는 패션 추천시스템
제작이 프로젝트 주제입니다. 이 시스템은 실생활 패션 관리, 온라인 쇼핑몰 추천, 스타
일 및 코디 컨설팅 등 다양한 분야에서 활용 가능하며, 특히 실생활에서 쉽게 활용 가능
한 실용적 앱입니다.

2. 프로젝트 목적 및 최종예상결과물
프로젝트 목적
- 현대인의 패션 선택 과정은 다양성과 복잡성으로 인해 많은 시간과 노력이 필요
합니다. 본 프로젝트는 AI 기반 이미지 분석과 텍스트 이미지 매칭 기술을 활용
하여, 사용자가 업로드한 옷 이미지를 자동으로 분석하고, 최적의 코디를 추천함
으로써 패션 선택 과정을 효율적으로 지원합니다.
- 프로젝트에서는 YOLOv5와 CLIP 두 개의 서로 다른 오픈소스 모델을 활용함으로
써, 학습, 탐지, 추천 기능을 모두 포함한 완전한 실용 앱을 개발할 예정입니다.
- 날씨 및 계절 정보, 사용자의 성격 유형(MBTI), 텍스트 기반 검색 등 다양한 
요소를 통합하여 개인화된 코디 추천을 제공하며, 각 추천에 대한 명확한 이유와
롤모델 정보도 함께 제공합니다.

최종 예상 결과물
1. 웹/데스크탑 UI: 사용자가 옷 이미지를 업로드하고 추천 코디를 확인할 수 있는
직관적인 인터페이스 제공
2. 이미지 분석 기능: YOLOv5 모델을 사용하여 상의, 하의, 신발 등 개별 옷 아이템
탐지
3. 스타일 분석 및 추천 기능: CLIP 모델을 사용하여 탐지된 아이템의 색상, 패턴, 스
타일을 분석하고, 데이터셋 기반으로 최적 코디 조합 추천
4. 추천 결과 시각화: 추천된 코디 이미지와 설명, 색상과 스타일 키워드 제공
5. 계절 및 날씨 기반 코디 추천: 사용자가 입력한 온도, 날씨, 계절 정보에 맞는 
코디 추천 (0번 기능)
6. 텍스트 기반 코디 검색: "파티용 코디", "출근룩", "데이트룩" 등의 텍스트 조건으로 
코디 검색 및 추천 기능 (1번 기능)
7. 추천 이유 제공: 각 코디 조합이 어울리는 이유를 MBTI, 계절, 날씨 등 다양한 
요소를 종합하여 명확하게 제시 (2번 기능)
8. 롤모델 제공: 추천된 코디와 관련된 연예인 사진 및 모범 코디 사례 제공 (3번 기능)
9. SNS 트렌드 분석: SNS 분석 및 이미지 수집을 통한 유행 스타일 반영 및 이번 
시즌 인기 코디 제공 (4번 기능)
10. MBTI 기반 개인화 추천: 사용자의 성격 유형(MBTI)을 반영한 개인화된 코디 추천
(예: ENFP → 자유롭고 컬러풀한 캐주얼 스타일, ISTJ → 깔끔하고 단정한 포멀 스타일) (5번 기능)
11. 코디 관련 화장법 제공: 추천된 코디에 어울리는 메이크업 스타일 제안 (6번 기능)
12. 다중 코디 추천: 하나의 입력에 대해 여러 버전의 코디를 추천하여 사용자 선택권 제공 (7번 기능)
13. 성능 보고서: 탐지 정확도, 추천 품질, 모델 처리 속도 등의 지표를 포함한 테스
트 결과 보고서 제공
14. 확장 가능성: 이후 기능 확장 시, 고급 개인화 알고리즘, 실시간 트렌드 반영 등 
추가 기능 구현 가능

3. 프로젝트에 활용할 오픈소스 개요 (오픈소스 소개, 활용 목적)
YOLOv5
- 객체 탐지 분야에서 널리 사용되는 딥러닝 모델로, 이미지 내 객체 위치와 클래
스를 빠르고 정확하게 탐지 가능. 학습·추론 속도가 빠르고, 다양한 커스텀 데이
터셋에 적용 가능
- 사용자가 업로드한 옷 이미지에서 상의, 하의, 신발 등 개별 아이템을 탐지

CLIP
- OpenAI에서 개발한 모델로, 이미지와 텍스트를 동일한 임베딩 공간으로 매핑하
여 의미 기반 검색, 분류, 이미지-텍스트 매칭 가능
- YOLOv5로 탐지된 옷 아이템의 스타일, 색상, 패턴 분석 → 데이터셋 기반 코디
추천
- 텍스트 쿼리("파티용 코디", "출근룩" 등)를 입력받아 유사한 스타일의 코디를 
검색하는 텍스트 기반 검색 기능 제공

활용 방법 상세
1. YOLOv5로 업로드된 옷 이미지에서 각 아이템 영역 탐지
2. 탐지된 아이템 이미지를 CLIP 모델에 입력하여 스타일 및 색상 분석
3. 사용자 입력 정보(온도, 날씨, 계절, MBTI)를 종합하여 개인화된 추천 필터링
4. 텍스트 쿼리("파티용 코디" 등)를 CLIP 모델로 처리하여 텍스트-이미지 매칭 수행
5. 데이터셋 내 유사 스타일의 다른 아이템과 매칭 → 최적 코디 후보 생성
6. SNS 트렌드 데이터를 분석하여 인기 코디 정보 반영
7. 여러 버전의 코디 추천 결과와 함께 추천 이유, 롤모델, 화장법 제안 제공
8. 추천 코디를 시각화하여 UI에 출력
